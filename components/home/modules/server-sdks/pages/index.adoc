= Ost Server SDKs

Ost server SDKs provide various methods that can be used to design and manage token economies. The SDKs handle multiple authentication scenarios allowing and caching, key management, and nonce management ensure that the overall end-user experience is smooth.

Ost robust RESTful APIs can also be used.
